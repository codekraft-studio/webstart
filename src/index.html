webstart

Simple webpack based seed project.

What is it?

A webpack based project with an optimized configuration for websites development that aim to be a solid starting point for your projects.

How it works?

During the development mode a webpack-dev-server is started with hot module replacement enabled and code source maps for a better debugging. The website can be accessed via port 8080 otherwise via port 8088 you can access BrowserSync server for cross-browser compatibility testing.
